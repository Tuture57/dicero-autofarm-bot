## Milestone Summary

### Key Accomplishments
1. Capture d'écran stable via PIL ImageGrab + DPI awareness — pixels identiques bot/templates
2. Script capture_templates.py entièrement revu : menu interactif, capture individuelle, calibration cartes/dés séparée
3. Détection des 11 états du jeu par template matching (9 initiaux + chest + edit_btn)
4. Clics fiables dans MirrorTo via SetCursorPos + SendInput sans MOUSEEVENTF_VIRTUALDESK
5. Auto-élévation admin, sortie propre par Échap, focus console restauré après clic

### Key Decisions
- SetCursorPos + SendInput sans VIRTUALDESK : seule méthode passant dans MirrorTo (testé 8 méthodes)
- Admin requis : UIPI de MirrorTo bloque les clics synthétiques sans élévation
- edit_btn détecté avant select_skill dans detect_state (partage de pixels)
- Calibration séparée cartes/dés : deux écrans différents du jeu

### Tech Debt
- Pas de détection de boucle infinie (même état pendant N cycles)
- Pas de screenshot debug en état "unknown" persistant
- Vérification du niveau de vie (< 10 = ne pas lancer) non implémentée
